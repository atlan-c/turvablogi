[[reply_to_current]]Hyvä kysymys — tämän jälkeen etenet näin:

1. **Luo uusi postaus**  
2. Avaa luotu `.md`-tiedosto ja kirjoita sisältö  
3. Paneelissa paina **Aja build** (varmistus)  
4. Paneelissa **Julkaise nyt** (suojattu: branch + PR-linkki)  
5. Paina **Avaa PR**  
6. GitHubissa: **Create pull request → Merge pull request**  
7. Paneelissa: **Avaa Actions** tai **Tarkista live-sivu**

Eli lyhyesti:  
**Luo → kirjoita → build → julkaise → PR/merge → tarkista live**.
